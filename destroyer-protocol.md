# Destroyer Protocol — 5-Kill Checklist

Every Destroyer agent runs this automatically before signing off on any finding.

## 1. Source check
Where does this number come from? Trace it. If the source is the same analysis that produced the claim, it's circular.

## 2. Cherry-pick check
Was this the best result selected from many attempts? What were the other attempts? If only the best is shown, the finding is overstated.

## 3. Confirmation bias check
Do we WANT this to be true? If yes, triple the scrutiny. The 33 Hz kill happened because we wanted it. The alpha scaling survived because we wanted it — check it again.

## 4. Regression check
Does this new finding contradict anything already on the site? The DMN regression (shower creativity vs meditation suppression) went unnoticed for 39 sessions.

## 5. Alternative explanation
What's the boring explanation? If you can't name one, you haven't tried. Zero kill ATTEMPTS is dishonest.
